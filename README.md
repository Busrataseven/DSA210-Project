# AI & Data Science Job Market Analysis

## Project Overview
This project analyzes the key factors affecting salaries in the AI and Data Science job market.

The main goal is to understand how variables such as experience level and work type influence salary levels using exploratory data analysis and statistical testing.


## Dataset
The dataset contains 10,345 job postings with features such as:

- Job Title  
- Company Size  
- Experience Level  
- Salary  
- Country  
- Skills  

The dataset includes both categorical and numerical variables and contains no missing values, making it suitable for analysis.


## Methods
The following methods were used in this project:

- Exploratory Data Analysis (EDA)  
- Data Visualization (Seaborn & Matplotlib)  
- Hypothesis Testing:
  - ANOVA (for experience level)
  - T-test (for work type)


## Key Findings
- Experience level has a strong and statistically significant effect on salary  
- Higher experience leads to higher salary levels  
- Work type (remote, hybrid, onsite) does not significantly affect salary  
- Salary distribution is slightly right-skewed  


## Conclusion
The results indicate that experience is the most important factor in determining salary, while work type has a limited impact.

This suggests that companies prioritize experience over work arrangement when setting compensation.


## Limitations and Future Work
This analysis is based on a single dataset and does not include all possible factors affecting salary.

Future work could include:
- Company size and industry analysis  
- Skill-based salary comparison  
- Machine learning models for salary prediction
