# DSA210-Project

## Project Overview
This project analyzes factors affecting salaries in the AI and Data Science job market.

## Dataset
The dataset includes 10,345 job postings with features such as:
- job title
- company size
- experience level
- salary
- skills

## Methods
- Exploratory Data Analysis (EDA)
- Hypothesis Testing (ANOVA, t-test)

## Key Findings
- Experience level has a significant effect on salary
- Work type (remote, hybrid, onsite) does not significantly affect salary
- Salary distribution is slightly right-skewed

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
